Sobre

Desenvolvemos este projeto pois identificamos o grande obstáculo de superar a procrastinação, é um desafio que todos enfrentam diariamente , sendo um empecilho para que as pessoas alcancem seus sonhos e objetivos, suas metas, idealizamos este sistema devido a carencia de um sistema realmente focado na gestão de metas com enfase na produtividade, assunto que e de importancia para todos pois é a produtividade coletiva que move a economia de uma nação gerando oportunidades, empregos e dignidade .



Instruções Iniciais

Este sistema tem um funcionamento bem simples, e feito a gestão de metas com base nos requisitos, as metas e requisitos são adicionadas pelo usuario, para tornar o sistema mais dinamico, atraves da quantidade de requisitos e possivel se medir o desempenho, e possivel se adicionar todo tipo de meta, o importante e a auto reflexão e compreensão dos requisitos essências para o cumprimento de uma meta, quanto mais detalhado e melhor o requisito, maior e a possibilidade de se atingir uma determinada meta, esta plataforma parte do principio de que toda meta tem requisitos necessários para seu êxito, ou seja, gerir, e se capacitar a realizar estes requisitos, é essencial para obtenção do sucesso, o trabalho duro e constante e focado em uma meta, garantem resultados, mas cabe ao usuario definir os requisitos necessários para o sucesso de sua meta, o funcionamento desta plataforma sera efetivo quando requisitos são bem elaborados.



Visualizar Desempenho

Visualizar o desempenho permiti ver o percentual de requisitos alcançados para uma determinada meta onde se calcula atraves do somatório de requisitos concluídos dividido pelo total de requisitos em uma meta, assim é possivel acompanhar entender e analisar seu progresso, é necessario uma visualização constante da evolução para que cada usuario consiga se motivar e manter o foco e desenvolvimento constante.
